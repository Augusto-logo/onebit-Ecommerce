# Task

Utilizar o **Dev Tools** do navegador para modificar a largura da página e ajustar o componente [Inicio.tsx](../components/Inicio.tsx) para as larguras de tela:

- Galaxy S10 (360px)
- Ipad Mini (768px)
- laptop (1024px)
- desktop / laptop MDPI (1280px)
- desktop grande / laptop HDPI (1440px)
- FULL HD (1920px)

Lembre-se de utilizar as classes do Tailwind para configurar a largura dos containers, utilizando as media queries (sm: / md: / lg: / xl: / 2xl:).

**Lembre-se**: Ao passar o mouse por cima de cada classe de **media query** do Tailwind, ele mostra a largura que aquela classe representa. Utilize isso para configurar as larguras dos containers.

Utilize a [documentação](https://tailwindcss.com/docs/responsive-design) de design responsivo caso necessário.

obs: O que quero é que você utilize o **Dev Tools** para manusear as larguras e veja onde o site(no caso o componente Inicio) quebra. Uma dica é remover os outros componentes para facilitar a visualização. **MAS**, sempre faça um teste final com todos os componentes.
